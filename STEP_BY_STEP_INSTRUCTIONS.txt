## üöÄ Method 1: Easiest Way (Recommended for Beginners)

### Step 1: Navigate to the Folder
1. Open **File Explorer** (Windows key + E)
2. Navigate to: `C:\Users\Soft\Desktop\Dev3\Whisper downloads`
3. You should see files like `start.bat`, `interactive_transcriber.py`, etc.

### Step 2: Launch the Tool
1. **Double-click** on `start.bat`
2. A black command window will open with a menu
3. Type `1` and press **Enter** to start Interactive Mode

### Step 3: Follow the Interactive Guide
The tool will ask you questions. Here's what to expect:

**Question 1: Video file path**
- Type the full path to your video file
- Example: `C:\Users\Soft\Videos\my_video.mp4`
- Or drag and drop your video file into the window
- Press **Enter**

**Question 2: Model size**
- You'll see options 1-5
- Type `2` for 'base' model (recommended)
- Press **Enter**

**Question 3: Output format**
- You'll see options 1-3
- Type `1` for 'txt' format (human-readable)
- Press **Enter**

**Question 4: Keyword search**
- Type `y` if you want to search for keywords, `n` if not
- Press **Enter**

**If you chose keywords:**
- Type your first keyword (e.g., "artificial intelligence")
- Press **Enter**
- Type your second keyword (e.g., "machine learning")
- Press **Enter**
- Press **Enter** again with no text to finish

**Question 5: Output location**
- Type `y` to save in the same folder as your video
- Press **Enter**

### Step 4: Wait for Processing
- The tool will show progress messages
- First time: Downloads AI model (~150MB)
- Then: Extracts audio from video
- Finally: Transcribes and searches for keywords
- **Total time: 3-10 minutes depending on video length**

### Step 5: Get Your Results
When complete, you'll find these files next to your video:
- `your_video_transcript.txt` - Full transcript with timestamps
- `your_video_search_results.json` - Keyword matches (if you searched)

## üõ†Ô∏è Troubleshooting Common Issues

### "File not found" Error
**Problem:** Can't find your video file
**Solution:** 
1. Check the file path is correct
2. Use full path: `C:\Users\YourName\Videos\video.mp4`
3. Make sure file extension is included (.mp4, .avi, etc.)

### "Permission denied" Error
**Problem:** Can't write to output folder
**Solution:**
1. Choose a different output folder (like Desktop)
2. Run as Administrator: Right-click `start.bat` ‚Üí "Run as administrator"

### Very Slow Processing
**Problem:** Taking too long to process
**Solution:**
1. Use smaller model: Choose option 1 (tiny) instead of 2 (base)
2. Split long videos into shorter segments
3. Close other programs to free up memory

### Poor Transcription Quality
**Problem:** Text doesn't match what's said
**Solution:**
1. Use larger model: Choose option 4 (medium) or 5 (large)
2. Improve audio quality if possible
3. Check if language is English (tool is optimized for English)

### "Module not found" Error
**Problem:** Python can't find required packages
**Solution:**
1. Make sure you're in the right folder
2. Use the virtual environment: `.\.venv\Scripts\activate`
3. Restart and try Method 1 (Interactive Mode)

## üìÅ Understanding Your Output Files

### Transcript File (TXT format)
```
[00:00:12 - 00:00:18] Welcome everyone to today's meeting about artificial intelligence.
[00:00:18 - 00:00:25] We'll be discussing the latest developments in machine learning algorithms.
[00:00:25 - 00:00:32] First, let's review the quarterly progress on our AI initiatives.
```

**Format explanation:**
- `[00:00:12 - 00:00:18]` = Start time - End time
- Text = What was said during that time period

### Search Results File (JSON format)
When you search for keywords, you get a detailed JSON file with:
- Exact timestamps for each keyword match
- Context words around each match
- Highlighted keywords for easy spotting

### CSV Format (for spreadsheets)
If you choose CSV output, you can open it in Excel with columns:
- Start Time
- End Time
- Formatted Start
- Formatted End  
- Text

## üéØ Pro Tips for Best Results

1. **Start Small:** Try a short 1-2 minute video first to test everything works
2. **Good Audio:** Clear, loud audio gives much better transcription results
3. **Specific Keywords:** Search for specific terms, not common words like "the" or "and"
4. **File Naming:** Use descriptive names for your videos for easier organization
5. **Save Space:** Delete the temporary audio files after processing (they're automatically cleaned up)
6. **Model Choice:** 
   - Use 'tiny' for quick tests
   - Use 'base' for most videos (good balance)
   - Use 'medium' or 'large' for important/difficult audio
